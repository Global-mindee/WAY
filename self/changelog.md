## 변화 유형 6종 설명

| 유형 | 의미 |
|------|------|
| 신규 | 기존에 없던 항목이 자기 정의에 처음 추가됨 |
| 갱신 | 기존 항목의 내용이 새 데이터로 교체됨 |
| 강화 | 기존 항목의 확신도가 올라감 (단서 1건 → 교차 확인) |
| 약화 | 기존 항목의 확신도가 내려감 (도전 데이터 등장) |
| 이동 | 항목이 Archive로 격리됨 (never_delete — 폐기가 아니라 Tier 이동) |
| 충돌 | 선언(Declared) vs 관찰(Observed) 불일치 발생 → conflicts.md 연계 |
